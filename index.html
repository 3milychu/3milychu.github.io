Playing with Hours of Music Played
Northside Festival

Course: Data Aesthetics and Information Visualization
Instructor: Christian Swinehart

This web app explores time and its behaviors as physical symbols. Data regarding venues, artists and shows listed for Brooklyn's 2017 Northside Festival was scraped by the instructor. I connected the respective JSON files, parsed by shows per venue, created arrays to store duration of all shows played per venue, and visualized the result as a simple rectangle shape which, upon hover, returns the number of hours played at each venue in number of 360 degree counter-clockwise spins.

Recent Earthquakes Investigator
USGS Earthquake Hazards Program

This web app was created as an exercise in using public data to make meaningful spatial representations of data. I used USGS Earthquake Hazard Program's data feed of earthquakes occuring in the last 30 days to an interactive map using Leaflet. The slider was created using P5.js by running a forloop that selected earthquakes within 0.04 magnitude of the slider value and updated the data points accordingly. The goal of this exercise was to pinpoint the location of each earthquake's epicenter meaningfully at each zoom level and give users an interactive interface to discover an interesting earthquake of their choice.

Explosive Remants of War (ERW) Zones in Kosovo

Course: Geographic Information Systems
Instructor: Stephen Metts

These maps were created using ArcMap from ArcGIS, using administrative, landmines and cluster bomb shapefiles. Using geoprocessing tools, areas of interest were clipped and intersected for analysis. Final maps used Python to calculate percentage of contamination as symbology.
