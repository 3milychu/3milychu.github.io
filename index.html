Schedules
Northside Festival, 2017

Methods & Goals

Course: Data Aesthetics and Information Visualization
Instructor: Christian Swinehart

This web app explores time and its behaviors as physical symbols. Data is sourced from Brooklyn's 2017 Northside Festival. Data analysis was performed in RStudio. The visualized result returns a simple rectangle shape which, upon hover, returns the number of hours played at each venue in number of 360 degree counter-clockwise spins.

Earthquakes
USGS Earthquake Hazards Program

Methods & Goals

This web app was created as an exercise in using public data to make meaningful spatial representations of data. I used USGS Earthquake Hazard Program's data feed of earthquakes occuring in the last 30 days to an interactive map using Leaflet. The slider selects earthquakes within 0.04 magnitude of the slider value. Users are able to dive into an interesting earthquake down to the neighborhood of origin.

Explosive Remants of War (ERW) Zones in Kosovo
Did we forget?

Methods & Goals

Course: Geographic Information Systems
Instructor: Stephen Metts

These maps were created using ArcMap from ArcGIS, using administrative, landmines and cluster bomb shapefiles. Using geoprocessing tools, areas of interest were clipped and intersected for analysis. Final maps used Python to calculate percentage of contamination as symbology.
