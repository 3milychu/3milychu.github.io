Time Play

Time Play
Northside Festival, 2017

Course: Data Aesthetics and Information Visualization
Instructor: Christian Swinehart

This web app explores time and its behaviors as physical symbols. Data is sourced from Brooklyn's 2017 Northside Festival. Data analysis was performed in RStudio. The visualized result returns a simple rectangle shape which, upon hover, returns the number of hours played at each venue in number of 360 degree counter-clockwise spins.
