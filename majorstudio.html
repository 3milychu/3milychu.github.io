This project analyzed 155k objects containing the Metropolitan Musem of Art's top seven occuring materials. Questions asked: What are representative items made from each material in the MET's collection? What larger patterns are they a part of, and what can we learn from the semiology of cultural artifacts?

Clustering Fashion with K-means
Major Studio MSDV 2018

Using machine learning, this project fits features to a k-means model to cluster objects from the Metropolitan Museum of Art's Costume Institute. The project focuses on understanding what happens when different cluster thresholds are set for the dataset, and allows views to zoom into centroids of interest.

Prototype
Process


1) Image Processing

Process images in the dataset for machine learning. This method used hierarchical segmentation with a threshold of 80 segments. This technique first identifies blocks of dominant color, condenses the image so that it is composed of those segments, and then converts it to greyscale with an exaggerated outline by adjusting the entropy.


2) Save Image Representations

After comparing the before and after transformation to adjust towards the desired result, the image transformations were then converted into 2d arrays of pixel values. This was done for all 7.6k images and saved as a flat feature set for later use.


3) Run K-means

K-means was selected as the model to cluster images based on similarity of shapes. The model was trained on creating 56 clusters from the dataset. Displayed are 3 example centroids from the model.


4) Join results to MET dataset and convert to JSON

The results of the kmeans model, with labelling of data by clusters and centroids, were joined to the original MET Costume Institute dataset. This was converted to JSON in preparation for data visualization.
