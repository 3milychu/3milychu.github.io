Clustering Images with K-Means

Disentangling Form: Using machines to draw geometric patterns in what we wear
Major Studio MSDV 2018

This project clustered items from the Metropolitan Museum of Art's Costume Institute by image similarity using k-means as a machine learning unsupervised task. The project focuses on understanding how items are clustered when extracting form from other factors such as color, year or size.

Web App in Progress
.

Process

Extracting Form: B&W Edge Detection

To "focus" only on form, I extracted the edge outlines of each image. This ignores color, dimensions and all other metadata such as date of creation, title of object and origins. The method used simply employs sobel edge detection to draw outlines around a black and white version of the image. It then blurs and accentuates detected edges and downscales the image.

Prepare Machine Learning Features

To prepare features for machine learning, each image after transformation is processed as a 15x35 pixel-by-pixel list representation of the image. This information is stored against the image name for easy retrieval and data joining in later steps.

Run K-Means

K-means was selected as the model to cluster images based on similarity of shapes. The model was trained on creating 56 clusters from the dataset. Displayed are 3 examples from three different clusters from the model.

Prepare Data for Visualization

After joining the feature representations to the original MET dataset containing metadata for all 7.6k items, a flat json was generated.

From here, the flat json was converted into a hierarchical parent-child json for data visualization in d3.

Each item is grouped a child of their parent cluster. Metadata preserved for display purposed includes: the title, link, src path, year, medium and culture for each item.

Analysis and Visualization

I began with filtering the visualization by model clusters, or groups of items that clustered quite well by the criteria I was aiming for: similarity in geometric form.

I added the capability to see quick details about each item in a cluster on hover, and a more detailed view that showed the original image of each item, or the metadata I pulled out when creating the hierarchical json.

Examining the model clusters helped me formulate a hypothesis on when extracting form is useful when grouping objects in this dataset together.

Storytelling & Narrative

To guide readers through my analysis, hypothesis and conclusions, various levels of interaction are embedded based on the viewer's literacy.

The web app begins with highlighted findings.

At any point, the viewer can scroll down to view a summary video on the project and its goals and findings.

Finally, the viewer can explore the complete cluster map of the Costume Institute in two ways: 1) the interactive view or 2) the annotated view.
