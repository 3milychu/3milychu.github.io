Explosive Remants of War (ERW) Zones in Kosovo
ArcMaps Infographic, 2017

Course: Geographic Information Systems Instructor: Stephen Metts These maps were created using ArcMap from ArcGIS, using administrative, landmines and cluster bomb shapefiles. Using geoprocessing tools, areas of interest were clipped and intersected for analysis. Final maps used Python to calculate percentage of contamination as symbology.
