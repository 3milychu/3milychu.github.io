Interactive Mapping

Find That Recent Earthquake
USGS Earthquake Hazards Program, 2017

This web app was created as an exercise in using public data to make meaningful spatial representations of data. I used USGS Earthquake Hazard Program's data feed of earthquakes occuring in the last 30 days using the interactive map library Leaflet. The slider selects earthquakes within 0.04 magnitude of the slider value. Users are able to dive into an interesting earthquake down to the neighborhood of origin.

Launch in Desktop Only
